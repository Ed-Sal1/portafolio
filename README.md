# Portafolio Profesional – Edgar Saltillo Ramírez

Este repositorio contiene el código fuente de mi **portafolio profesional como Desarrollador Java Full Stack Jr.**, cuyo propósito es presentar de forma clara, estructurada y visualmente coherente mi perfil, experiencia, habilidades técnicas y proyectos. El portafolio está diseñado como una **aplicación web de una sola página (one-page)**, priorizando la navegación fluida, la lectura rápida y una jerarquía visual adecuada, aspectos clave tanto para evaluación académica como para procesos de reclutamiento.

El proyecto fue desarrollado utilizando **HTML5, CSS3 y JavaScript (ES6)**, sin el uso de frameworks de frontend, con el objetivo de demostrar dominio sólido de las bases del desarrollo web y un control directo sobre la estructura, los estilos y el comportamiento de la interfaz. La arquitectura es modular y escalable, lo que permite integrar nuevas secciones, proyectos o mejoras visuales sin afectar el funcionamiento general del sitio.

## Objetivo del portafolio

El objetivo principal de este portafolio es:
- Comunicar de manera efectiva mi perfil profesional y técnico.
- Mostrar proyectos reales y académicos con un enfoque en impacto y resolución de problemas.
- Facilitar a reclutadores y evaluadores una revisión rápida y clara de mis competencias.
- Servir como base evolutiva para futuras mejoras y ampliaciones conforme avance mi carrera profesional.

## Estructura del proyecto

El portafolio está organizado de la siguiente manera:

- **Inicio (Hero):** Presentación personal, rol profesional y enlaces clave de contacto.
- **Acerca de mí:** Descripción de mi perfil, motivaciones y enfoque profesional.
- **CV:** Resumen profesional con opción de descarga del currículum en formato PDF.
- **Tecnologías:** Visualización de tecnologías utilizadas mediante iconografía.
- **Portafolio de proyectos:** Sección dinámica que muestra proyectos destacados, rol desempeñado, tecnologías y enlaces relevantes.
- **Contacto:** Formulario de contacto y datos profesionales para establecer comunicación directa.

## Tecnologías utilizadas

- HTML5  
- CSS3 (variables CSS, diseño responsivo, microinteracciones)  
- JavaScript (ES6)  
- Git y GitHub para control de versiones  
- Iconografía SVG para tecnologías  
- Diseño responsive y buenas prácticas de accesibilidad  

## Características destacadas

- Diseño one-page con navegación por anclas.
- Renderizado dinámico de proyectos mediante JavaScript.
- Estilos basados en variables CSS para fácil personalización de paleta de colores.
- Microinteracciones y animaciones suaves controladas por JavaScript.
- Diseño responsivo adaptable a dispositivos móviles, tabletas y escritorio.
- Enfoque en claridad visual y escaneo rápido de información.

## Despliegue

El portafolio puede ser desplegado en cualquier servidor estático (por ejemplo, GitHub Pages o Netlify), ya que no requiere dependencias adicionales ni procesos de compilación.

## Estado del proyecto

El portafolio se encuentra **en desarrollo activo**. La estructura base está completa y funcional, pero el contenido y los estilos continuarán evolucionando conforme se integren nuevos proyectos, aprendizajes y mejoras visuales.
